Overview

What is the 10DLC Chrome Extension?

The extension helps Salesforce users quickly review outbound SMS text content for alignment with 10DLC expectations. It is simple, fast and private. Your text messages are not saved or stored anywhere.

Privacy-first: The extension analyzes text you select or the message in view. It returns a clear status with issues and tips.
How it works

Two Ways to Run a Check

This is the core experience. Pick the flow that matches where you are working.

Opt-in assumed

1) Floating Pill (Salesforce)

  • Bottom-right button labeled Check 10DLC appears in Salesforce.
  • Click to check the message currently in view.
  • Stop instructions and Help tips are removed — you’re allowed to text the merchant in this context.
Opt-in unknown

2) Right-Click Menu (Anywhere)

  • Highlight any text → right-click → Check 10DLC.
  • Works inside or outside Salesforce, on any web page.
  • Stop and Help tips are shown when opt-in isn’t guaranteed.
What you’ll see

Results You’ll See

  • Status: Pass · Warning · Fail
  • Issues: Concise reasons, no internal rules exposed
  • Tips: Practical suggestions to improve your message
Opt-in assumed = Pill mode hides Stop/Help lines  ·  Opt-in unknown = Right-click mode includes Stop/Help lines.
Access

Installation & How to Open this Help

  1. Install the extension (via your admin or the provided CRX/Chrome Web Store link).
  2. After install, you’ll see the floating Check 10DLC pill in Salesforce and the right-click menu everywhere.
  3. To open this Help page later:
    • Right-click the extension icon → Options → Help / Documentation.
Troubleshooting

Quick Fixes

  • Don’t see the pill in Salesforce? Refresh the page. Ensure you’re on a Salesforce tab and the extension is enabled.
  • Right-click item missing? Select some text first, then right-click. Reopen Chrome if needed.
