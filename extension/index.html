How it works

Three ways to run a check

Opt-in assumed

Floating Pill in Salesforce

A bottom-right Check 10DLC pill appears in Salesforce. Click to review the message in view. In this trusted context, Stop/Help lines are hidden.

  • Works in Lightning and Classic
  • One-click review of the current message
Opt-in unknown

Highlight text ‚Üí Right-click

Highlight any text, right-click, choose Check 10DLC Compliance. Works inside or outside Salesforce.

  • Supported in Lightning and Classic
  • Great for ad-hoc checks on any web page
Universal

Open the panel from the extension icon

Click the 10DLC Checker extension icon in Chrome to open the panel on any page. Paste or type your message to analyze it directly.

  • Works on any site, no selection required
  • Use when you want to draft and refine copy
