Registration Process #

  • Brand registration: legal name, EIN, address
  • Campaign registration: use case, sample messages, opt‑in proof

Approval times vary by provider and manual review needs.

Compliance Essentials #

  • Consent only message opted‑in recipients
  • Identity clearly state the sender/brand
  • STOP/HELP support common keywords
  • Links avoid public shorteners in initial outreach
  • Restricted content follow carrier codes of conduct

Examples #

✅ Compliant Welcome

"Welcome to BrandName alerts! Msg&data rates may apply. Up to 3 msgs/week. Reply STOP to opt out."

❌ Risky First Message

"50% OFF today only! Click bit.ly/xxxx" — lacks brand identity, no STOP, public short link.

FAQs #

Do I need 10DLC if I use short code or toll‑free?

Not necessarily; each channel has separate verification and suitability.

Which carriers enforce this?

All major U.S. carriers (AT&T, Verizon, T‑Mobile) enforce registration and content rules for A2P long code messaging.
